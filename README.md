# ❤️ AI Girlfriend - Emma

A simple and private AI Girlfriend built using **Streamlit** and **Groq API**.

---

## ✨ Features

- Beautiful chat interface
- Personalized AI girlfriend (Emma)
- Sweet, playful, and affectionate personality
- Remembers your conversations (Memory saved locally)
- Fast responses powered by Groq
- 100% private chat history (saved on your computer)
- Easy to customize name and personality

---

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **AI Backend**: Groq API (Llama 3.3 70B)
- **Memory**: 2D List + JSON file
- **API Key Management**: `.env` file

---

## 📁 Project Structure
AI_Girlfriend/
├── app.py                 # Main Streamlit application
├── .env                   # Groq API Key (Do not share)
├── emma_memory.json       # Saved chat history (auto-generated)
├── requirements.txt       # List of dependencies
└── README.md              # Project documentation
